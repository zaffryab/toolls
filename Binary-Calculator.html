Understanding Binary Numbers

What is Binary?

Binary is a base-2 numeral system that uses only two digits: 0 and 1. It's the fundamental language of computers.

Binary Arithmetic

Basic operations:

  • 0 + 0 = 0
  • 0 + 1 = 1
  • 1 + 1 = 10 (carry 1)
  • 1 - 1 = 0
  • 10 - 1 = 1 (borrow)

Binary Applications

  • Computer programming
  • Digital electronics
  • Data storage
  • Networking protocols
  • Cryptography

Binary Prefixes

  • Kibibyte (KiB) = 1024 bytes
  • Mebibyte (MiB) = 1024 KiB
  • Gibibyte (GiB) = 1024 MiB
  • Tebibyte (TiB) = 1024 GiB
