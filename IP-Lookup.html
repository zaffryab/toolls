How to Use the IP Lookup Tool

1

Enter an IP Address

Type any public IP address into the search field. You can enter IPv4 (e.g., 192.168.1.1) or IPv6 addresses.

2

Lookup Information

Click "Lookup" to retrieve detailed information about the IP address, including location and network details.

3

Check Your Own IP

Click "Use My IP Address" to automatically lookup information about your current public IP address.

4

Review Results

View comprehensive details including geographical location, ISP information, and security data.
