Color Format Guide

HEX

Hexadecimal color codes start with # followed by 6 characters (0-9, A-F) representing red, green, and blue values.

RGB

Red, Green, Blue values each ranging from 0 to 255. Example: rgb(74, 108, 247)

HSL

Hue (0-360), Saturation (0-100%), Lightness (0-100%). A more intuitive way to describe colors.

HSV

Hue (0-360), Saturation (0-100%), Value (0-100%). Similar to HSL but with different calculations.
