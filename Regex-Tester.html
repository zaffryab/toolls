How to Use This Regex Tester
1

Enter Your Regex Pattern

Type your regular expression pattern in the "Regular Expression Pattern" field. For example: [A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Za-z]{2,} to match email addresses.

2

Provide Test Text

Enter the text you want to test against your regex pattern in the "Test String" field. You can type or paste any text content.

3

Select Options

Choose any regex flags you need: Ignore Case (i), Global Match (g), Multiline (m), or Dot All (s).

4

Run the Test

Click the "Test Regex" button to execute your regular expression against the test string.

5

Analyze Results

Review the results section to see if your pattern matches, view any errors, and inspect the matched content.

Regex Quick Reference

Basic Patterns

  • . - Any character
  • \d - Digit
  • \w - Word character
  • \s - Whitespace

Quantifiers

  • * - 0 or more
  • + - 1 or more
  • ? - 0 or 1
  • {n} - Exactly n

Anchors

  • ^ - Start of string
  • $ - End of string
  • \b - Word boundary
