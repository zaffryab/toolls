How to Use This Tool

1 Text to Morse

Enter your text in the left box and click the right arrow button (→) to convert it to Morse code.

2 Morse to Text

Enter Morse code (using dots and dashes) in the right box and click the left arrow button (←) to convert it to text.

3 Clear

Use the clear button (×) to reset both input and output fields.
