How to Use the UUID Generator Tool

This tool allows you to generate universally unique identifiers (UUIDs) with various formatting options.

Generating UUIDs

  1. Select the UUID version (v4 for random, v1 for time-based)
  2. Choose how many UUIDs you want to generate (1-100)
  3. Select your formatting options:
    • Uppercase letters: Display letters in uppercase
    • Include hyphens: Include hyphens in the standard format
    • Include braces: Wrap the UUID in curly braces
    • URL-safe encoding: Use base64url encoding (for v4 only)
  4. Click the "Generate UUIDs" button
  5. Copy individual UUIDs or all of them at once using the copy buttons

About UUIDs

A UUID (Universally Unique Identifier) is a 128-bit number used to uniquely identify information in computer systems.

UUID Versions

  • v1 Version 1 (Time-based): Generated using timestamp and MAC address
  • v4 Version 4 (Random): Generated using random or pseudo-random numbers

UUID Format

A standard UUID is represented as 32 hexadecimal digits, displayed in five groups separated by hyphens, in the form 8-4-4-4-12 for a total of 36 characters (32 alphanumeric characters and 4 hyphens).

Example: 123e4567-e89b-12d3-a456-426614174000

Common Uses

  • Database keys as primary identifiers
  • Session identifiers in web applications
  • Message tracking in distributed systems
  • Unique filenames and resource identifiers
  • Tracking items in inventory systems

Collision Probability

The probability of a duplicate UUID is extremely low. To have a 50% chance of one collision, you would need to generate 2.71 quintillion (2.71×10¹⁸) version-4 UUIDs.
