How to Use
  1. Paste your JSON Web Token (JWT) in the input field above.
  2. Click the "Decode Token" button to parse and display the token components.
  3. The decoded header, payload, and signature will appear in the results section.
  4. Use the "Clear" button to reset the input and results.
  5. Click "Load Sample" to load a sample token for testing.
  6. JWTs consist of three parts: Header, Payload, and Signature separated by dots.
  7. This tool helps you debug and inspect JWTs without verifying the signature.
About JWT

JSON Web Token (JWT) is an open standard (RFC 7519) that defines a compact and self-contained way for securely transmitting information between parties as a JSON object.

This information can be verified and trusted because it is digitally signed. JWTs can be signed using a secret (with the HMAC algorithm) or a public/private key pair using RSA or ECDSA.
